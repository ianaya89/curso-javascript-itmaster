Internet

  • Proyecto militar principios de los 60
  • Red de redes, conexion entre computadoras
  • Protocolos, los idiomas de las computadoras (forma de comunicación)
  • Protocolos para mails, chats, archivos...

TCP

  • Transmission Control Protocol
  • Armar paquetes de datos y distribuirlos al destino correcto.
  • Equipos que escuchan para 'hablar' con otros equipos usando puertos (servidores)
  • Equipos que llaman a los 'escuchadores' conectandose a estos puertos (clientes)

WWW

  • Creada por Tim Bernes-Lee
  • Conjunto de protocolos y formatos para navegar en sitios webs mediante browsers
  • Sistema de distribución de documentos de hipertexto conectados atraves de enlaces
  • Protocolo HTTP (80) o HTTPS (443)

HTTP

  • Orientado a transacciones, esquema de peticiones y respuestas
  • Cada respuesta obtenida es un recurso y cada recurso tiene una URL

URL

Protocolo + dominio y puerto + path

http://mysite.com/index.html

Cada servidor tiene una IP.

HTML

  • Formato de documentos usado para páginas web
  • Lenguage de marcado (XML)
  • Establece estructura y código para la definicion de contenido
  • Basado en referenciación

HTML

  • Etiquetas + Atributos + Texto
  • head: información del documento.
  • body: documento en sí.
          
            <!doctype html>
            <html>
            <head>
              <title></title>
            </head>
            <body>

            </body>
            </html>
Etiqueta script nos permite incluir JavaScript en nuestro HTML o hacer referencia a un archivo externo con el atributo src. Ubicar script antes de cerrar body.

CSS

  • Lenguage de estilos mediante reglas combinadas
  • Define la presentación de un documento HTML (parte visual)
  • Estilos en linea con atributo style
  • Estilos en HTML con equitea style
  • Estilos en archivos externos con equitea link
  • Elementos + Clases + ID's

window.location

Nos permite accerder a informacion de la URL

          
            location.href; // URL de la página
            location.hostname; // Nombre del dominio
            location.pathname; // Nombre del path
            location.protocol; // Nombre del protocolo

window.navigator

Contiene información sobre el browser que esta visitando nuestro sitio.

          
            navigator.userAgent; // Informacion del browser.
            navigator.platform; // Nombre del S.O.
            navigator.language; // Devuelve el idioma y la region del browser
            navigator.product; // Nombre del motor de JS

window.history

Nos permite acceder al historial del browser.

          
            history.go(10); // 10 pasos/paginas adelante
            history.go(-10); // 10 pasos/paginas atras
            history.back(); // Pagina anterior
            history.foward(); // Pagina siguiente

Document Object Model

  • Representación estructural del documento (árbol)
  • Conjunto de objetos que pueden ser manipulados usando JavaScript
  • Cada elemento HTML es un nodo
  • Permita modificar contenido y presentación visual
  • Forma de comunicar scripts con HTML

Estructura

document

Nos permite acceder al documento y sus nodos.

document.documentElement representa la raiz del DOM.

Nodo raiz, nodos padres, hijos y hermanos

  • parentNode
  • childNodes
  • firstChild
  • lastChild
  • previousSibling
  • nextSibling

Ejemplos

Encontrando Elementos

Nos devuelven un arreglo de objetos (o un objeto). Algunos de estos métodos nos devuelven objetos vivos.

  • document.getElementsByTagName('nombreTag')
  • document.getElementById('idTag')
  • document.getElementsByClassName('claseTag')
  • document.querySelectorAll('selector')
  • document.querySelector('selector')
          
            Ejemplo

Cambiando Elementos

Métodos para mover, borrar o ubicar elementos.

  • document.removeChild
  • document.addChild
  • document.appendChild
Los nodos solo pueden estar en UN lugar.

Creando Elementos

Métodos para crear nuevos elementos.

  • document.createTextNode
  • document.createElement

Atributos de los Elementos

Cada elemento HTML tiene atributos y estos pueden ser modificados.

  • elemento.nombreDeAtributo
  • elemento.getAttribute('nombre')
  • elemento.setAttribute('nombre', 'valor')

Estilando Elementos

Podemos cambiar, agregar o eliminar reglas de estilos mediante el atributo style.

style es un objeto que tiene cada elemento donde podemos hacer 1 o más declaraciones.

La manera correcto de estilar consiste en agregar, cambiar o eliminar clases de CSS que contengan definidas dichas reglas.

  • elemento.style
  • elemento.className

Eventos

Son la forma que tenemos de darle a nuestro sistema la posibilidad de reaccionar a determinadas acciones y manejar esas reacciones mediante código.

Tipos de Eventos

  • elemento.addEventListener('nombreEvento', function(){})
  • elemento.removeEventListener('nommbreEvento')

Formularios

Podemos accederlos atraves del array forms del objecto document.

Cada form contiene un arreglo de fields (se pueden acceder por índice o clave).

          
            function validateForm() {
                var email = document.forms['contact']['email'].value;
                if (email) {
                    alert('Email must be filled out');
                    return false;
                }
            }

setTimeout

Espera X cantidad de tiempo para ejecutar una funcion.

          
            setTimeout(function(){ 
              alert('Hello); 
            }, 3000);

setInterval

Ejecuta una funcion de manera iterativa cada X cantidad de tiempo.

          
            setInterval(function(){ 
              alert('Hello'); 
            }, 3000);
