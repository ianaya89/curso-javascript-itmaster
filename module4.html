Qué es jQuery?

Es una libreria creada por John Resig que nos facilita y simplicafica la manera de manipular e interactuar con el DOM

  • Encontrar y manipular elementos HTML
  • Manejar Eventos
  • Efectos Visuales

Write less, do more.

Implementación

Se agregar referencia al archivo que contiene la libreria

          
            <script src="http://code.jquery.com/jquery-2.1.4.min.js"></script>
            <script src="js/jquery-2.1.4.min.js"></script>
Se puede tener el archivo local o hacer uso de una CDN Agregar referencia despues de nuestros CSS y antes de nuestros scripts

Fundamentos

Concepto de Wrapper

Los elementos se jquerizan.

element !== $(element)

$

jQuery provee la funcion global $ o jQuery que nos permite encontrar elementos HTML usando selectores de CSS.

          
            $('p');
            $('.item')
            $('#container')

            $('body p a')
            $('.list li')
Es multifacetica: acepta selectores CSS, string en formato HTML y variables. Uso similar a document.querySellectorAll()

$(document).ready

Este evento nos permite saber cuando nuestro DOM esta listo para ser manejado. Se dispara una vez que todos los scripts y css hayan sido cargados y el renderizado de la página haya terminado.

          
            $(document).ready(function(){
              //...
            });

            $(function(){
              //...
            });
Anterior al window.load Podemos declarar N $(document).ready.

Encadenamiento

Las funciones de jQuery retornan (casi siempre) el elemento que se obtiene en el selector.

         
           $('#p1').css('color', 'red').slideUp(2000).slideDown(2000);

           $('#a').hide().fadeIn(2000);

No Conflict

  • Liberar el usa de la variable $
  • Evitar futuros problemas
  • Customizar name spaces
         
           var ITMaster = jQuery.noConflict();

Plugins

  • Utilidad para ampliar las funcionalidades del framework
  • Resuelven problemas específicos y son reutilizables
  • No es mas que una funcion que se le extiende al objeto jQuery

Creando un Plugin

        
          jQuery.fn.desaparece = function() {
             this.each(function(){
                elem = $(this);
                elem.css('display', 'none');
             });   
             return this;
          };

          $('h1').desaparece();
