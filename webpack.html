Webpack

Module Bundler

  • Separar y cargar On Demand
  • Mantiene el tiempo de carga
  • "Todo" puede ser un modulo

Setup

CLI

npm install --save-dev webpack

Bundlear

webpack ./entry.js bundle.js

webpack.config.js

Anatomía Básica

            
              module.exports = {
                entry: './src/index.js',
                output: {
                  path: __dirname,
                  filename: 'lib/index.js'
                },
                module: {
                  loaders: [
                    { test: /\.css$/, loader: 'style-loader!css-loader' },
                    { test: /\.js$/, loader: 'babel-loader' }
                  ]
                }
              }

Loaders

Que Son?

Modulos para cargar modulos NO JS

babel-loader

npm install --save-dev babel-core babel-preset-es2015 babel-loader

webpack.config.js

            
              // ...

              module: {
                 loaders: [{
                     test: /\.js$/,
                     exclude: /node_modules/,
                     loader: 'babel-loader'
                 }]
             }
