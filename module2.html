Variables

Es un contenedor para un valor. Se declaran por medio de la palabra clave var pero no se indica el tipo de dato.

Una misma variable puede almacenar diferentes tipo de datos (no simultaneamente) durante su ciclo de vida.

          
            var num = 15;
            var chars = 'cat';
            var char2 = 'dog;'
            var $ = 'jQuery';
            var _ = 'underscore';

            /* Ejemplos no permitidos

            var -temp = 'some value';
            var 2num = 2;

            */
Los nombres usados para variables pueden contener letras, numeros o simbolos, pero deben comenzar con letras, '_' o '$'.

Boolean

Representan 2 valores: true y false.

Operadores

  • Asignación
  • Aritméticos
  • Comparación
  • Lógicos
  • Unarios

Asignación

Asigna un valor al operador de la izquierda basado en el valor de la derecha.

          
            var x = 1,
            y = 2;

            x += y; // x = x + y
            x -= y; // x = x - y
            x *= y; // x = x * y
            x /= y; // x = x / y
  • Más Operadores

Aritméticos

Resuelven operaciones aritmeticas y devuelven valores numericos.

  • Suma: +
  • Resta: -
  • Multiplicacion: *
  • Division: /
  • Resto: %
  • Incremento: ++
  • Decremento: --

Aritméticos

          
            var opt = 100 + 4 * 11; // 144
            var opt2 = (100 + 4) * 11; // 1144

            var opt3 = 10 - 7; // 3
            var opt4 = 1 - 2 + 1; // 0

            var opt5 = 12 / 3; // 4

            var opt6 = 10 % 2; // 0
            var opt7 = 11 % 2; // 1

            var opt8++; // opt8 = opt8 + 1;
            var opt9--; // opt9 = opt9 - 1;

Comparación

Comparan los valores y devuelven un valor logico basado en el resultado de la comparación.

  • Igual: ==
  • No Igual: !=
  • Estrictamente Igual: ===
  • Estrictamente No Igual: !==
  • Mayor: >
  • Mayor o Igual: >=
  • Menor: <
  • Menor o Igual: <=

Comparación

          
            var comp1 = 1 == 1; // true
            var comp2 = 1 != 1; // false

            var comp3 = 1 == '1'; // true
            var comp4 = 1 === '1'; // false      
            var comp5 = 1 !== '1'; // true

Lógicos

Proporcionan valores booleanos (no siempre) al evaluar un determinada condición.

  • AND
  • OR
  • NOT
          
            var and1 = true && true; // false
            var and2 = true && false; // false
            var and3 = false && true; // false
            var and4 = false && false; // false

            var or1 = true || true; // true
            var or2 = true || false; // true
            var or3 = false || true; // true
            var or4 = false || false; // false

            var not1 = !true; // false
            var not2 = !false; // true

Unarios

Operadores que utilizan un solo valor.

  • typeof
  • delete
  • void
          
            typeof undefined; // 'undefined'
            typeof null; // 'object'
            typeof true; // 'boolean'
            typeof false; // 'boolean'
            typeof 'hi'; // 'string'
            typeof 10 // 'number'
            typeof 1.6 // 'number'

Estructuras Condicionales

  • if - else
  • switch

if - else

Se ejecuta el codigo definido dentro del bloque if si la condición es verdadera. Si la condición es falsa se ejecutara el else (en caso de que exista).

if - else

          
            if (true === true) {
              //...
            }

            if (1 !== 1) {
              //...
            }
            else {
              //...
            }

            var str = 'hi';
            if (str.length === 2) {
              //...
            }
            else if (str.length === 3) {
              //...
            }
            else {
              //...
            }

truthy and falsy

          
            if (true) { } // true
            if (1) { } // true
            if ('some value') { } // true

            if (0) { } // false
            if (false) { } // false
            if ('') { } // false
            if (null) { } // false
            if (undefined) { } // false

Condicional Ternario

condicion ? valor1 : valor2;

          
            var ageLimit = 18;

            var status = ageLimit === 18 ? 'adult' : 'minor';

switch

Hace comparación estricta de un valor y ejecuta el/los casos que tiene definido/s para ese valor.

          
            var value = 4;

            switch(value) {
              case 1:
              //...
              break;

              case 2:
              case 4:
              //...
              break;

              default: 
              //...
            }

Estructuras Ciclicas

  • while
  • do-while
  • for
  • for in

while

Se ejecuta mientras la condicion sea verdadera.

        
          var count = 1;

          while (count < 5) {
            //...
            count++;
          }

do-while

Se ejecuta mientras la condicion sea verdadera, pero se garantiza al menos 1 ejecución.

        
          var count = 6;

          do {
            //...
          } while (count < 5)

for

3 partes:

  • Define el inicio del bucle: var i = 0
  • Se establece la condición: i < 10
  • Expresión que incrementa un valor: i++
        
          for (var i = 0; i < 10; i++) {
            //...
          }

for in

Permite recorrer objetos, arrays o strings a traves de sus propieades.

        
          var str = 'hola',
          str2 = '';

          for (var s in str) {
            str2 += s;
          }

          // str2 = 'hola'

Funciones

Una función es un conjunto de instrucciones que se agrupan para realizar una tarea concreta y que se pueden reutilizar fácilmente.

  • Nombre la función (opcional)
  • Argumentos de la funcion (opcional)
  • Cuerpo de la funcion
        
          function add(x, y) {
            if (x && y ) {
              return x + y;
            }

            return 0;
          }

          add(1, 2); // 3

Funciones como variables y argumentos

Una función puede ser asignada a una variable, e incluso puede ser usada como parametro en otra función. Esto se puede hacer ya que las funciones son objetos.

        
          var add = function(x, y) {
            //..
          };

          var subtract = function (x, y) {
            //...
          };

          function addAndSubtract(x, y, Z, add, subtract) {
            var a = add(x, y);
            var b = add(x, z);

            return subtract(a, b);
          }

Funciones Autoinvocadas

Permite ejecutar una funcion inmediatamente despues de ser definida.

        
          (function load(){
            //...
          })();

          (function add(x, y){
            //...
          })(1, 2);

Hoisting

Independientemente de donde esté la declaración de una variable, ésta es movida al inicio del ámbito al que pertenece (En tiempo de ejecucion).

        
          function foo() {
            alert(x);
            var x = 10;
          }

          function foo() {
            var = x;
            alert(x);
            x = 10;
          }

Scope

Es el ámbito y contexto de una variable.

El ámbito de una variable es el lugar dentro de un programa en el cual dicha variable vive y por lo tanto, donde puede ser usada.

Significa que una variable definida dentro de una función (con la palabra reservada var) va a poder ser usada dentro de esa función, pero no fuera de la misma.

Las variables declaradas fuera de una funcion (o sin la palabra var) son denomidas globales y seran accesibles en todos lados.

Scope

        
          var value = 'global';

          function showValue() {
            alert(value); //global;
          }
        
      
        
          var value = 'global';

          function showValue() {
            var value = 'private';
            alert(value); //private;
          }
        
      
        
          function showValue() {
            alert(value); // Error;
          }

Closure

Podes definir funciones dentro de otras funciones, pudiendo acceder desde las funciones mas internas al scope de las mas externas (no al reves).

        
          function incrementAndAdd(x, y) {

            function increment() {
              x++;
              y++;
            }

            return x + y;            
          }

          incrementAndAdd(5, 5); // 12

Object

  • Tienen propiedades (caracteristicas) y metodos (comportamiento).
  • Son conjuntos de pares clave-valor.
  • Si no es primitivo, es un objeto.
  • Un objeto puede contener otro objeto.
  • Son Mutables

Instanciación

        
          // No usar instanciación con new Object().

          var obj = { // Notación literal.
            prop1: 'Hello World!',
            method: function() {
              //..
            }
          };

          obj.prop2 = 'Bye World!';

Acceso a Propiedades

        
          var Person = {};
          
          person.name = 'John';
          person.id = 14;

          person.sayHello = function() {
            return 'Hello my name is: ' + this.name;
          }
          
          alert(person.id);
          alert(person['id']);

          alert(person.sayHello());

Object.prototype

Todos los objetos de JavaScript estan linkeados a un objeto prototype.

Nos sirve para definir y luego extender comportamiento hacia otras clases (herencia).

Usando Constructores

        
          var Person = function(name, id) {
            this.name = name;
            this.id = id;
          }

          Person.prototype.sayHello = function() {
            return 'Hello my name is: ' + this.name;
          }

          var john = new Person('John', 14);
          var jane = new Person('Jane', 15);

          alert(john.sayHello());
          alert(jane.sayHello());

Herencia Pseudoclassical

        
          // Objeto Padre
          var Person = function() {
              this.canTalk = true;
          };

          // Definicion de metodos
          Person.prototype.greet = function() {
            if (this.canTalk) {
                console.log('Hi');
            }
          };
          
          // Objeto Hijo 1
          var Employee = function(name, title) {
              this.name = name;
              this.title = title;
          };

          // Aplicacion de Herencia
          Employee.prototype = new Person();

          // Definicion de metodos
          Employee.prototype.greet = function() {
            if (this.canTalk) {
                console.log('I am ' + this.name);
            }
          };
  
          // Objeto Hijo 2
          var Customer = function(name) {
              this.name = name;
          };
          
          // Aplicacion de Herencia
          Customer.prototype = new Person();
          
          // Instancias
          var bob = new Employee('Bob','Constructor');
          var joe = new Customer('Joe');
          var rg = new Employee('John','Developer');
          var mike = new Customer('Mike');
    
          bob.greet();
          joe.greet();
          rg.greet();
          mike.greet();

          // Objeto Padre

Array

Estuctura de memora lineal donde los datos son accedidos mediante indices.

Son menos eficientes que los arrays de otros lenguages de programacion porque son objetos.

El primer indice de un array es 0.

Instanciación

        
          // No usar instanciación con new Array().
          
          var empty = []; // Notación literal.

          var nums = [1, 2, 3, 4, 5];
          var chars = ['a', 'b', 'c', 'd'];
          var mix = [1, 'a', {name: 'John'}, [4, 5, 6]];

Acceso y Métodos

        
          var nums = [1, 23, 56, 78, 81];

          nums[0]; // 1

          nums.push(99);
          nums.length; // 6

          nums.pop();
          nums.length; // 5

          delete nums[2]; // [1, 23, undefined, 78, 81];

          nums.splice(2, 1); [1, 23, 78, 81];

Tips de Sintaxis

  • No se tienen en cuenta los espacios en blanco ni los saltos de linea (Es importante el indentado)
  • Es Case Sensitive
  • Usamos camelCase (salvo en los constructores)
  • No se define el tipo de dato de las variables
  • No es obligatorio el ';' al final de cada sentencia (Pero hagan de cuenta que si)
  • Comentarios en linea con // Comentario
  • Comentarios multilinea con /* Comentario */
  • Se pueden usar tanto comillas simples como comillas dobles

Ejemplos

          
            // Espacios y Saltos de linea
            function sum(x,y){if(x&&y){return x+y;}return 0;}

            function sum(x, y) {
              if (x && y) {
                return x + y;
              }

              return 0;
            }

            'hello' !== 'HELLO'; // Case Sensitive
            person !== Person;

            // Comentario en linea

            /*

              Comentario Multilinea

            */

            'hello' === "hello";
